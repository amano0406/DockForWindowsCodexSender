# Product note: DockForWindowsCodexSender

This repository is a send-only bridge for Windows Codex / Codex CLI.

v0.1 scope:

- repo registry
- prompt templates
- render prompt
- render all prompts
- dry-run send
- send one prompt via Codex CLI
- send log

Out of scope for v0.1:

- Windows app UI automation
- response scraping
- autonomous scheduling
- automatic merge
- destructive operations

Prioritize:

- dry-run
- send logs
- one-repo send before send-all
- safe CLI
- clear errors
