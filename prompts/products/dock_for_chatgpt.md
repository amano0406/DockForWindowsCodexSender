# Product note: DockForChatGPT

DockForChatGPT is a bridge/tool product, not a Timeline product.

Role:

- ChatGPT Web operation support
- prompt submission support
- thread retrieval support
- Deep Research support
- Timeline output review support

Rules:

- Do not treat DockForChatGPT as source of truth.
- Do not expose secrets, cookies, or sessions.
- Do not perform external actions without explicit confirmation.
- Expect Web UI changes to break automation.
